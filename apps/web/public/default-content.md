# Live Editor - 使い方ガイド

Live Editorは、ブロックベースのマークダウンエディタです。各ブロックをクリックして編集できます。

## 主な特徴

### WYSIWYG（What You See Is What You Get）

編集中もマークダウンのレンダリング結果がリアルタイムで表示されます。マークダウン記法を意識せずに、見たままの形式で文書を作成できます。編集していないブロックは常にプレビュー状態で表示されます。

### ブロックベース編集

各段落や要素が独立したブロックとして管理されており、個別に編集・移動が可能です。直感的な操作感を実現しています。

## 基本操作

### キーボードショートカット

- `Enter`: 新しいブロックを作成
- `Backspace`: 空のブロックを削除して前のブロックに統合
- `Esc`: 編集を終了
- `↑` / `↓`: ブロック間の移動（カーソルが端にある時）
- `Tab` / `Shift+Tab`: インデント調整（リスト・コードブロック内）
- `Cmd+Z` / `Ctrl+Z`: 元に戻す（Undo）
- `Cmd+Shift+Z` / `Ctrl+Shift+Z`: やり直し（Redo）

### ドラッグ&ドロップ

各ブロックの左側にあるハンドルをドラッグして、ブロックの順序を自由に変更できます。

## フォーマットツールバー

編集中のブロックの上に表示されるツールバーで、以下のフォーマットを適用できます：

- **見出し**: H1、H2、H3
- **テキスト装飾**: 太字、斜体、インラインコード
- **リスト**: 箇条書き、番号付き、チェックリスト
- **その他**: 引用、コードブロック、テーブル

## サポートしているマークダウン記法

### 見出し

`#`の数で見出しレベルを指定します（H1〜H6）。

### テキスト装飾

通常のテキストに加えて、**太字**、*斜体*、`インラインコード`が使えます。

### リスト

#### 箇条書きリスト

リスト内でEnterキーを押すと新しいリスト項目が追加されます。空のリスト項目でEnterキーを押すとリストから抜けます。

- リスト項目1
- リスト項目2
  - ネストされた項目
  - もう一つのネスト項目
- リスト項目3

#### 番号付きリスト

1. 最初のステップ
2. 次のステップ
3. 最後のステップ

#### チェックリスト

チェックボックスをクリックするだけでチェック状態を切り替えられます。

- [ ] 未完了のタスク
- [x] 完了したタスク
- [ ] 進行中のタスク

### 引用

引用ブロック内でEnterキーを押すと新しい引用行が追加されます。ネストも可能です。

> "The only way to do great work is to love what you do."
>
> — Steve Jobs

### コードブロック

コードブロックでは、言語に応じたインデント（Python: 4スペース、JavaScript: 2スペースなど）が自動適用されます。Tab/Shift+Tabでインデントの調整も可能です。

```javascript
function calculateSum(a, b) {
  const result = a + b;
  return result;
}

const total = calculateSum(10, 20);
console.log(total);
```

```python
def calculate_sum(a, b):
    result = a + b
    return result

total = calculate_sum(10, 20)
print(total)
```

### テーブル

テーブルはスラッシュコマンドまたはツールバーから作成できます。横スクロールに対応しているので、列数が多くても問題ありません。

#### スラッシュコマンド

`/table 3 4` のように入力して、スペースキーまたはEnterキーを押すと、3行4列のテーブルが自動生成されます。

| 機能 | 対応状況 | 備考 |
|------|---------|------|
| 見出し | ✅ 完了 | H1〜H6対応 |
| リスト | ✅ 完了 | ネスト対応 |
| コード | ✅ 完了 | シンタックスハイライト |
| テーブル | ✅ 完了 | スラッシュコマンド・横スクロール対応 |
| ドラッグ&ドロップ | ✅ 完了 | ブロック並び替え |
| Undo/Redo | ✅ 完了 | カーソル位置も復元 |

### URL自動リンク化

URLを入力してフォーカスを外すと、自動的にマークダウンリンク形式に変換され、リッチプレビューカードが表示されます。

## コピー機能

### 全体コピー

右上の「Copy Markdown」ボタンをクリックすると、編集内容全体がマークダウン形式でクリップボードにコピーされます。適切な空行も自動で追加されます。

### セクションコピー

見出し（H1〜H6）にマウスをホバーすると、右側にコピーボタンが表示されます。このボタンをクリックすると、その見出しから次の同レベル以上の見出しまでのセクション全体がコピーされます。ホバー時には、コピー対象のセクションが青色でハイライト表示されます。
